## 🚀 第二部分：操作步骤

### 步骤一：获取订阅链接

1. 登录 **CrossGo** 官网后台，在"捷径"栏点击 **「一键订阅」**。

   ![点击一键订阅按钮](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/1.png)

2. 在弹出的菜单中选择 **「复制订阅地址」**。

   ![复制订阅地址](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/2.png)

### 步骤二：将订阅导入 FlClash

1. 打开 FlClash 客户端，点击左侧导航栏的 **「配置」** (文件夹图标)。
2. 点击右下角的 **「+」** 号按钮。

   ![点击加号按钮](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/3.png)

3. 在弹出的"添加配置"菜单中，选择 **「URL」**。

   ![选择 URL 选项](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/4.png)

4. 在"从 URL 导入"框中，**粘贴** 刚才复制的订阅地址，然后点击 **「提交」**。

   ![粘贴订阅链接并提交](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/5.png)

### 步骤三：选择连接节点

1. 导入成功后，点击左侧导航栏的 **「代理」** (文档图标)。

   ![点击代理图标](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/6.png)

2. 在展开的节点列表中，**点击选择** 一个您想使用的节点（如：自动选择、香港、美国等）。

   ![选择节点](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/7.png)

### 步骤四：开启代理服务

1. 点击左侧导航栏最上方的 **「仪表盘」** (方格图标)。
2. **关键步骤：** 将 **「系统代理」** 的开关打开（变为蓝色状态）。
3. 点击右下角的 **「播放」** 按钮，正式启动服务。

   ![开启系统代理并启动服务](https://cdn.jsdelivr.net/gh/crossgo-cc/Tutorial@main/FlClash/Windows/assets/8.png)

## ✅ 第三部分：连接确认与常见问题

### 1. 如何确认连接成功？

- 在 **「仪表盘」** 页面，观察左上角的 **“网络速度”** 是否有流量曲线波动。
- 查看 **“网络检测”** 处的 IP 是否已变为海外地址。

### 2. 模式说明

> **💡 建议选择「规则」模式：**
>
> - **规则 (Rule)：** 智能分流，海外网站走加速，国内网站（如百度、淘宝）走直连，省流量且速度快。
> - **全局 (Global)：** 所有网络访问都经过代理。
> - **直连 (Direct)：** 不使用代理。

### 3. 无法上网怎么办？

- 检查是否开启了 **「系统代理」** 开关。
- 尝试在“代理”页面切换不同的节点。
- 检查电脑系统时间是否准确（时间不同步会导致连接失败）。
